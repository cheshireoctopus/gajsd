Browser Events

⇒ click

⇒ mouse over

⇒ mouse out

⇒ key down

⇒ key up

⇒ form submit

• We can use JavaScript to catch and respond to events

• More event info

Browser Events

Handle events in three steps:

1) Use the jQuery selector, $(), to select the element(s) you want to apply the event to:


	$('#my-awesome-btn')

2) Call the desired event method on the selected element


	$('#my-awesome-btn').click()

3) Pass the event a callback function; this function will run when the event fires


	$('#my-awesome-btn').click(function (event) {
		console.log(event);
	})

Browser Events

• We can attach multiple events to the same element


  $('#my-box').mouseenter(function () {
  	$('#my-box').css('background', 'red');
  })

  $('#my-box').mouseleave(function () {
  	$('#my-box').css('background', 'blue');
  })

Browser Events

• $(this) is handy for listening to events on many elements of the same type or class

• Refers to whatever element caused the event to trigger

• JavaScript's this is wrapped with jQuery's $(); enables jQuery methods on vanilla this


	$('.my-list-item').click(function () {
	  $(this).addClass('active');
	  $(this).siblings().removeClass('active');
	})

• Codepen Example

Browser Events

• Direct events (seen previously), are only attached to existing DOM elements on page load

• Events on dynamically created elements must be set differently:


	$(document).on('.my-list-item', 'click', function () {
	  $(this).addClass('active');
	  $(this).siblings().removeClass('active');
	})

HTML Manipulation

HTML Manipulation

.append('content')

⇒ Insert "content" at the END of each element


	$('div').append('<p>Appended paragraph</p>')

.prepend('content')

⇒ Insert "content" at the BEGINNING of each element


	$('div').prepend('<p>Prepended paragraph</p>')
HTML Manipulation

• Assuming the following HTML:


	<div id="someDiv">
	  <p>A paragrpah of text</p>
	</div>

• The following JS:


	$('#someDiv').append('<p>Appended paragraph</p>');
	$('#someDiv').prepend('<p>Prepended paragraph</p>');

• Will yield the following HTML:


	<div id="someDiv">
	  <p>Prepended paragraph</p>
	  <p>A paragrpah of text</p>
	  <p>Appened paragraph</p>
	</div>
HTML Manipulation

.remove()

⇒ Remove (delete) the selected element


	$('#someDiv').remove()

.empty()

⇒ Remove all content from INSIDE an element; do not delete the element


	$('#someDiv').empty()

JavaScript Templates

• JS templates afford a degree of separation between logic (JS) and presentation (HTML)

• Help avoid excessive DOM manipulation

• Consider using once you are generating HTML from your JS

• Data ⇒ Template ⇒ View

• Many templating libraries:

⇒ handlebars.js

⇒ mustache.js

⇒ underscore.js

Handlebars.js

• Handlebars.js has a four step process:

1) Create the template structure

2) Compile the template

3) Pass the compiled template a JavaScript data structure

4) Append the template to the DOM

Handlebars.js

1) Create the template structure:

• Use <script> tags to create the template

• Must have an id and type="text/x-handlebars-template"

• Write HTML inside the template, and surround dynamically generated data with {{}}


  <script id="title-template" type="text/x-handlebars-template">
    <h1>{{title}}</h1>
    <p>{{description}}</p>
  </script>
Handlebars.js

2) Use jQuery to reference the newly created template and compile using Handlebars.compile()


	var source = $('#title-template').html();
	var template = Handlebars.compile(source);

3) Pass a data object to the newly created template

4) And add it to the DOM via jQuery.


	var titleObject = {
	  title: 'Presidents',
	  description: 'A JSD Project'
	};

	// 3
	var titleTemplate = template(titleObject);

	// 4
 	$('#title').append(titleTemplate);
Handlebars.js

• Handlebars provides the #each helper for array iteration

• When iterating over an array, use {{this}} to reference each element


  <script id="title-template" type="text/x-handlebars-template">
    {{#each presidents}}
      <option>{{this}}</option>
    {{/each}}
  </script>
